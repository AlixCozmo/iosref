--- layout: default title: Arrays cheatsheet (Swift 4) ---

Array

```swift var myArr = [1, 3] myArray.append(5) ```

Loop over array

```swift for element in myArr { print("Element: \(element)") } /* Output: Element: 1 Element: 3 Element: 5 */ ```

Map

```swift let squaredArr = myArr.map { $0 * $0 } print(squaredArr) // Output: [1, 9, 25] ```

Filter

```swift let over2 = myArr.filter { $0 > 2 } print(over2) // Output: [3, 5] ```

Sorted

```swift let arr = [4, 28, 2, 1, 99, 25] let over2 = myArr.sorted() print(over2) // Output: [3, 5] ```
